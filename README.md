# Chat_PDF

Discutez avec vos PDFs 😎🤖

Chat_PDF est un projet Python qui vous permet d’interagir avec vos documents PDF via une interface conversationnelle. Téléchargez un PDF, posez-lui des questions et obtenez des réponses intelligentes basées sur le contenu de votre document !

## Fonctionnalités

- 🗂️ Téléchargement et traitement de fichiers PDF
- 💬 Interface de chat pour poser des questions sur votre PDF
- 🧠 Utilisation de techniques avancées de traitement du langage naturel (NLP)
- 🔒 Vos données restent privées – aucun stockage externe
- 🌐 Interface simple et conviviale

## Prise en main

### Prérequis

- Python 3.7+
- pip

### Installation

1. **Clonez le dépôt :**
    ```bash
    git clone https://github.com/mentalist011101/Chat_PDF.git
    cd Chat_PDF
    ```

2. **Installez les dépendances :**
    ```bash
    pip install -r requirements.txt
    ```

### Utilisation

1. **Lancez l’application :**
    ```bash
    python app.py
    ```

2. **Ouvrez votre navigateur à l’adresse suivante :**
    ```
    http://localhost:5000
    ```
3. **Téléchargez votre PDF et commencez la discussion !**
